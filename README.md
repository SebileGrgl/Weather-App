# Weather App

## Description

This is a simple weather application that allows users to check the current weather conditions for a specific city. It fetches data from the OpenWeatherMap API and displays information such as temperature, weather description, humidity, and wind speed. The application also provides visual representation of the weather through corresponding weather icons.

## Features

- **Search by City:** Users can enter the name of a city to retrieve its weather information.
- **Error Handling:** If the entered city is not found, the application displays an error message.
- **Dynamic UI:** The UI updates dynamically based on the retrieved weather data.
- **Animated Feedback:** The search box shakes when empty or when an invalid city is entered, providing visual feedback to the user.

## Technologies Used

- **HTML:** Markup for the application's structure.
- **CSS:** Styling for the user interface.
- **JavaScript:** Backend functionality and interaction with the OpenWeatherMap API.

## Usage

1. Enter the name of a city in the input field.
2. Click the search icon or press Enter to fetch weather data.
3. View the current weather conditions for the specified city.
